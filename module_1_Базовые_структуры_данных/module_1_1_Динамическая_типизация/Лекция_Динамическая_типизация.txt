Адрес памяти. Id объектов

В Python тип данных хранится не в переменной, а в самом объекте, на который она ссылается. Стоит упомянуть, что в
Python объектом является всё.
Как мы помним, переменная — это часть памяти, в которую мы помещаем объект для последующего использования.
В Python работать напрямую с адресами памяти невозможно. Однако есть одно исключение — функция id, которая в
определённых реализациях может показать нам адрес памяти объекта. Тем не менее, её основная и, пожалуй, единственная
цель — проверить, ссылаются ли два имени на один и тот же объект, так как обратиться по этому адресу средствами Python
невозможно.
